## About Solara

Solara is a free Roblox script executor made specifically for Windows. It gives users a simple way to inject and run Lua scripts inside Roblox sessions with a single click. The experience is intentionally minimal, with no required key system and no account signup step.

Included is a hand-picked script hub with more than 500 community scripts spanning automation, utility, and gameplay-oriented tools. Solara also ships with an auto-update system and a persistent injection queue so it can track Roblox changes while keeping the interface clean and quick to use. Whether you are trying custom code or browsing prebuilt options, Solara is designed to provide a dependable Lua execution setup on Windows 10 and 11.

**Is Solara safe to use?**  
Solara is offered as-is for educational and testing use. Anyone using it is fully responsible for their actions. Always inspect scripts before running them.

**Will Solara work after Roblox updates?**  
The auto-update engine tries to restore compatibility after Roblox patches, but success is not guaranteed. Refer to the repository for the latest update information.

**How does Solara compare to other executors?**  
Solara is built around a lightweight, no-key workflow with a встроенный script hub. It does not ask for registration or paid licensing.

**Can my Roblox account be penalized?**  
Using third-party script executors goes against Roblox's Terms of Service and can lead to account moderation. Use at your own discretion.

**Where are my scripts stored locally?**  
User scripts are stored in the `scripts/user/` directory inside the installation folder. Script hub content lives under `scripts/hub/`.
